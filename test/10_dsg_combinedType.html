Time Series of Profile(s)

Time Series of Profile(s)#

time series (station) of profile(s): Time series of profiles at fixed locations. A file can contain many stations and many time series at each station. Examples: profilers, balloon soundings.

  • Only one station in the file

  • Each station has the same number of profiles, and the same number of vertical levels for each profile

  • Each station has a different number of profiles AND/OR the level coordinates for each station may vary

Example Dataset: Newport Lab CTD Casts, 1997-2008
